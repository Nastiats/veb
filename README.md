лабораторная работа 1
"проектирование архитектуры динамического веб-приложения"

1. Тема: разрабтка сайта по продаже железнодорожных билетов.
Разработка системы электронной продажи билетов на поезда с использованием личного кабинета. Система позволяет регистрировать поезда, рейсы, количества мест на них. Также, осуществляет продажу билетов пассажирам.
Целевая аудитория: пассажиры железной дороги.
Проект будет иметь несколько режимов работы: пользовательский и системный. В системном режиме система должна позволять регистрировать поезда, связанные с ними рейсы, количество в каждом поезде вагонов разного типа, получать общий отчёт о количестве проданных билетов. В пользовательском режиме система должна отображать расписание отправления поездов, количество свободных мест различого типа, возможность их резервирования (приобритения), в личном кабинете должна быть возможность отображения приобритённых пользователем билетов.

2. Структура приложения:
  1.	Главная (/) – общая информация, возможность вызова странички аутенфикации;
  2.	Аутенфикация (/auth) – ввод логина и пароля, ссылка на станицы восстановления пароля (/auth/restore), регистрации нового пользователя (/auth/new);
  3.	Работа в личном кабинете:
    1.	Выбор поезда (номер, дата отправления, тип вагона) (/select);
    2.	Покупка билетов на выбранный поезд (/sale);
    3.	Просмотр списка приобретенных билетов с возможностью повторить приобретение билетов на другую дату (/mytickets);
  4.	Администрирование (/adm) – состоит из нескольких разделов: 
    1.	Аутентификация (/adm/auth);
    2.	Ввод типа вагона (/adm/vtype);
    3.	Формирование поезда (/adm/train);
    4.	Формирование расписания движения поездов (/adm/schedule);
    5.	Получение статистики проданных билетов (/adm/stat);

3. Технологический стек
  1.	На стороне браузера выполняется javascript приложение с использованием фреймворка  react.js ;
  2.	На стороне сервера выполняется приложение написанное на языке JAVA с использованием фреймворка Spring;
  3.	Для хранения данных используется СУБД PostgresSql.
